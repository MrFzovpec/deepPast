<head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <meta http-equiv="X-UA-Compatible" content="ie=edge"> <!-- Импортируем jQuery --> <script src="https://code.jquery.com/jquery-3.4.1.min.js" integrity="sha256-CSXorXvZcTkaix6Yvo6HppcZGetbYMGWSFlBw8HfCJo=" crossorigin="anonymous"></script> <!-- Импортируем стили deepPast --> <link rel="stylesheet" href="...path-to-dpast-folder/dpast/css/dpast.css"> <!-- Импортируем скрипты deepPast --> <script type="text/javascript" src="...path-to-dpast-folder/dpast/js/dpast.js"></script> <!-- Импорт Ваших стилей --> <link rel="stylesheet" href="css/main.css"> <title>Document</title> </head>

Оформление текста для обработки

Для выделения зоны с текстом используется родительский класс dpast, для выделения параграфа с текстом нужно использовать класс dpast__content

<div class="dpast"> <p class="dpast__content"> Унылая пора! Очей очарованье! <br> Приятна мне твоя прощальная краса — <br> Люблю я пышное природы увяданье,<br> В голубь и в золото одетые леса,<br> В их сенях ветра шум и свежее дыханье,<br> И мглой волнистою покрыты небеса,<br> И редкий солнца луч, и первые морозы,<br> И герой седой зимы угрозы.<br><br> А.С. Пушкин </p> </div>

Окно для вывода информации о слове/речевом обороте/словосочетании

Для добавлении окна, создайте блок с id dpast__description. В этот блок добавьте следущие элементы
1) Изображение или блок с id close__dpast__description
2) span с id dpast__description__title
3) Пустой параграф p с id dpast__description__content

<div id="dpast__description"> <img id="close__dpast__description" src="https://img.icons8.com/ios/40/000000/close-window.png"> <span id="dpast__description__title"></span> <br><br> <p id="dpast__description__content"></p> </div>

Как установить расширение

В первую очередь, нужно скачать и распаковать архив с расширением. После нужно открыть браузер Google Chrome и перейти на chrome://extensions/, включить режим разработчика и загрузить папку с расширением с помощью кнопки "Загрузить распакованное расширение". (См. видео ниже)
