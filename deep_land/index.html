О проекте

Наш проект представляет собой комплексное решение проблемы незнания значений слов/словосочетаний/речевых оборотов, которое обеспечивает быстрое и точное разъяснение устаревшей речи. В качестве продуктов мы предоставляем пользователю расширение для браузера и SDK (приемущественно для разработчиков).


Перейти к документации

SDK

1) Наша SDK библиотека дает возможность добавить на ваш сайт автоматическое распозниние уставревшего слова.
2) Наш SDK облегчит пользователям поиск значения интересющих их слов, ведь для этого не нужно будет открывать дополнительный сайт, достаточно лишь нажать на неизвестное слово.
3) Искусственный интелект с высокой точностью определяет устаревшее слово и выдает его значение.
4) Просто и удобно в использовании как для разработчика, так и для пользователя.

Расширение для Google Chrome

Расширение для браузера анализирует контент на странице и находит устаревшие слова, выводя их списком в удобном окне. Пользователю будут доступны определения всех слов.
